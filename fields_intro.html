What are fields

Fields are an important object in mathematics. We first learn about them in highschool. There we talk about the properties of natural numbers, integer, fractions and real numbers. The last two form a field. But at highschool you don't usually hear the formal definition of a field. Instead we learn that the addition and multiplication are commutative and associative, that all nonzero integers and real numbers have an multiplicative inverse and so on. If you don't study for a degree in mathematics then it might be possible that you use the field structure for one thing only and that is solving a system of linear algebraic equations. The coefficients by the unknows must be from a field. Why? That is a tough questions. Probably the best answer is that it works quite well and we like it. Let's look at an example

$$\begin{aligned} 3x+2y&=1 \\ x-y&=2 \\ \end{aligned}$$

What do we do when we try to solve this system of equations? First we move the \(y\) to the other side of the equation with an opposite sign to get \(x=2+y\). We substitute this expression to the first equation to obtain \(3(2+y)+2y=1\). We multiply the content in the brackets to get \((6+3y)+2y=1\). We move the brackets to obtain \(6+(3y+2y)\) and add the coefficients for the variable \(y\) to get \(6+5y=1\). We subtract the number 6 from both sides and divide by the number \(5\) to find out that \(y=-1\). We substitute the result to the second equation to get \(x=1\). What if we did not know that the fractions are distributative? Then we couldn't perform the third step and wouldn't be able to solve this system of equations! So we know that the field structure is useful for something. Let's look at the formal definition

Field is a set \(R\) with two binary operations \(\odot : R \rightarrow R \rightarrow R\) and \(\oplus : R \rightarrow R \rightarrow R\) and two distinct elements \(1 \in R\) and \(0 \in R\) such that they satisfy following axioms

  • \(\forall \, a,b,c \in R\) it follows that \((a \oplus b) \oplus c = a \oplus (b \oplus c)\)
  • \(\forall \, a \in R\) it follows that \(a \oplus 0 = a\)
  • \(\forall \, a \in R \enspace \exists \, b \in R\) such that \(a \oplus b = 0\)
  • \(\forall \, a,b \in R\) it follows that \(a \oplus b = b \oplus a\)
  • \(\forall \, a,b,c \in R\) it follows that \((a \odot b) \odot c = a \odot (b \odot c)\)
  • \(\forall \, a \in R\) it follows that \(a \odot 1 = a\)
  • \(\forall \, a \in R \enspace \exists \, b \in R\) such that \(a \odot b = 0\)
  • \(\forall \, a,b \in R\) it follows that \(a \odot b = b \odot a\)
  • \(\forall \, a,b,c \in R\) it follows that \(a \odot (b \oplus c) = a \odot b \oplus a \odot c\)

How does a mathematician think about this algebraic structure? For a mathematician the equal yet more useful definition is that field is a particular type of group and monoid with additional axiom of having a multiplicative inverse for all nonzero elements. This goal of this text is to build the necessary algebric structures in order to get to this definition and hopefully explain why a mathematician would remember the definition in such a manner.
