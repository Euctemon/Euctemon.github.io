Introduction

We can start with a definition of a monoid. You cannot have much less axioms as that. It might seem almost useless but we will that it gets handy in defining a ring.

Monoid is a set \(R\) with a binary operation \(\odot\) and an element 1 such that for all \(a,b \in R\) is \(a \odot b\) also in \(R\) and \(a \odot 1 = a\) and \(1 \odot a = a\).
