Writing proofs

Axiomatic systems

In the previous chapter, we discussed a way how to check satisfiability of a formula in a given language. We noted that for an infinite model, the truth tables come short. So what do mathematicians do instead? They write proofs. A proof is essentially a sequence of formulas. We start with the so called axioms and then use rules for constructing new formulas out of old ones, hoping that at the end we derive the formula we want to prove. The rules are commonly know as inference rules and together with set of axioms constitute an axiomatic system.

Axiomatic systems have a long history. Using set of axioms and inference rules dates back to Euclid and his elements. Euclid called the axioms of geometry as postulates. But the proper study of axiomatic systems can be said to begin with Frege and Hilbert. They were not only using logic for mathematics, but they were interested in the axiomatic systems itself. What follows is a version of an axiomatic system for propositional logic Hilbert was using.

Axioms
  1. \(\mathfrak{a} \to (\mathfrak{b} \to \mathfrak{a})\)
  2. \((\mathfrak{a} \to (\mathfrak{b} \to \mathfrak{c})) \to ((\mathfrak{a} \to \mathfrak{b}) \to (\mathfrak{a} \to \mathfrak{c}))\)
  3. \(\mathfrak{a} \to (\mathfrak{b} \to \mathfrak{a} \land \mathfrak{b})\)
  4. \(\mathfrak{a} \land \mathfrak{b} \to \mathfrak{a}\)
  5. \(\mathfrak{a} \land \mathfrak{b} \to \mathfrak{b}\)
  6. \(\mathfrak{a} \to (\mathfrak{b} \lor \mathfrak{a})\)
  7. \(\mathfrak{b} \to (\mathfrak{b} \lor \mathfrak{a})\)
  8. \((\mathfrak{a} \to \mathfrak{c}) \to (\mathfrak{b} \to \mathfrak{c}) \to (\mathfrak{a} \lor \mathfrak{b} \to \mathfrak{c})\)
  9. \(\lnot \lnot \mathfrak{a} \to \mathfrak{a}\)
Rules
  1. If we can prove \(\mathfrak{a}\) and \(\mathfrak{a} \to \mathfrak{b}\), we can prove \(\mathfrak{b}\).

We see that Hilbert's axiomatic system has only one rule called modus ponens. Philosophers use this implication rule for many types of logic and they sometimes call it modul ponendo ponens. This can be translated as method that by affirming, affirms. The rules we use for producing new formulas out of the proven ones are called inference rules, because we infer new things from old ones and the axioms are often called axiom schemas. This is because \(\mathfrak{a}, \mathfrak{b}\) and \(\mathfrak{c}\) are not concrete formulas, but rather symbols which can be substitued for any formula we want. We can for example instatiate the first axiom schema as \(p \to (q \to p)\) or \((p \land q) \to ((r \lor s) \to (p \land q))\) where \(p, q, r\) and \(s\) are variables for the propositional logic.

The interesting thing is that these axiom schemata were chosen so that every one of them produce tautology. You can check that with truth tables. Before we look at how we use such axiomatic system, we write down the definition of a proof.

Proof is a sequence \(\Psi_0,\ldots,\Psi_n\) of formulas such that for every \(k < n + 1\), either \(\Psi_k\) is a formula in context \(\Gamma\), an axiom schema or there are indices \(i_0,\ldots,i_l< k\) such that \(\Psi_k\) follow from \(\Psi_{i_0}, \ldots, \Psi_{i_l}\) by some inference rule. We say that \(\Psi_n\) is the conclusion and the sequence \(\Psi_0,\ldots,\Psi_n\) with hypotheses in \(\Gamma\) is a proof of \(\Psi_n\). We write \(\Gamma \vdash \Psi\) if there exists a proof of \(\Psi\) given the context \(\Gamma\).

This is one of the possible ways to define what a proof is. We will see another type of proof when we get to natural deduction. We notice that the definition features something called context. This is a set of formulas which we suppose are true in the context of a given proof. It is not always the case that \(p \lor q\) is true, but we can add it to the context and then build proofs upon this assumption. Suppose we choose a context made of \(c \lor b\), \(c \to d\) and \(b \to d\). The variable \(c\) stands for car, \(b\) for bus and \(d\) for staying dry.

\(\{c \lor b, c \to d, b \to d\} \vdash d\).
  1. Obtain \((b \to d) \to (c \to d) \to ((b \lor c) \to d)\) by instantiating the axiom schema 8.
  2. Obtain \((c \to d) \to ((b \lor c) \to d)\) by modus ponens using the context formula \(b \to d\) and the formula obtained in step 1.
  3. Obtain \((b \lor c) \to d\) by modus ponens using the context formula \(c \to d\) and the formula obtained in step 2.
  4. Finally obtain \(d\) by modus ponens using the context formula \(b \lor c\) and the formula obtained in step 3.

Supposing we have either a car or a bus at our disposal and that both car and bus will keep us dry, we concluded that we would stay dry. The context is important. We can't prove that we stay dry just because we wish so.

So far, we used concrete formulas in our proofs. Now let's take a formula \((p \lor q) \land r\). We would like to prove that it is equivalent to \(r \land (p \lor q))\). What we can do is that we can check whether \(s \land r\) is equivalent to \(r \land s\) and treat the variable \(s\) simply as a resulting truth value of the subformula \(p \lor q\). If someone gave is the formula \(s \land r\), we wouldn't know if the rewrite was made. This way, we blur the line between concrete formulas and schemata. Let's see an example.

\(\{\mathfrak{a} \land \mathfrak{b}\} \vdash \mathfrak{b} \land \mathfrak{a}\).
  1. Obtain \(\mathfrak{a}\) by modus ponens using the context and axiom schema 4.
  2. Obtain \(\mathfrak{b}\) by modus ponens using context and axiom schema 5.
  3. Obtain \(\mathfrak{a} \to (\mathfrak{b} \land \mathfrak{a})\) by modus ponens using \(\mathfrak{b}\) and axiom schema 3.
  4. Obtain \(\mathfrak{b} \land \mathfrak{a}\) by modus ponens using \(\mathfrak{a} \to (\mathfrak{b} \land \mathfrak{a})\) and \(\mathfrak{a}\).

We learned something about the conjugation symbol. It does not matter if we swap the position of formulas \(\mathfrak{a}\) and \(\mathfrak{b}\). We say that the conjunction is commutative. Let's prove something seemingly trivial.

\(\varnothing \vdash \mathfrak{a} \to \mathfrak{a}\).
  1. Obtain \(\mathfrak{a} \to (\mathfrak{a} \to \mathfrak{a})\) from the the axiom schema 1.
  2. Obtain \(\mathfrak{a} \to ((\mathfrak{a} \to \mathfrak{a}) \to \mathfrak{a})\) from the axiom schema 1.
  3. Obtain \((\mathfrak{a} \to ((\mathfrak{a} \to \mathfrak{a}) \to \mathfrak{a})) \to ((\mathfrak{a} \to (\mathfrak{a} \to \mathfrak{a})) \to (\mathfrak{a} \to \mathfrak{a}))\) from the axiom schema 2.
  4. Obtain \(\mathfrak{a} \to (\mathfrak{a} \to \mathfrak{a})) \to \mathfrak{a} \to \mathfrak{a}) \) by modus ponens using formulas from steps 2 and 3.
  5. Obtain \(\mathfrak{a} \to \mathfrak{a} \) by modus ponens using formulas from steps 4 and 1.

First thing you should note is that the context is empty. This means that the formula is true without any supplemental assumtions, which makes such a formula a tautology. You might ask why isn't this formula an axiom. It seems useful to know it from the start. We could add \(\mathfrak{a} \to \mathfrak{a}\) to the set of axiom schemata and nothing would break. But we strive to have a set of axioms which is in some sense minimal. This means that no axiom in the set is provable using the other axioms.

The following theorem summarizes some important properties of the proof system. Proofs of these statements follow easily from the definition of a proof and the assumption that the rules of the axiomatic system contain modus ponens.

The set of consequences

Suppose \(\Gamma\) is a set of formulas. The set of consequences is the set of formulas provable from \(\Gamma\) and it is inductively generated by the following rules:

  • If \(\mathfrak{a} \in \Gamma\), then \(\Gamma \vdash \mathfrak{a}\)
  • If \(\mathfrak{a}\) is an axiom schema, then \(\Gamma \vdash \mathfrak{a}\)
  • If \(\Gamma \vdash \mathfrak{a}\) and \(\Gamma \vdash \mathfrak{a} \to \mathfrak{b}\), then \(\Gamma \vdash \mathfrak{b}\)

One of the interesting properties of this proof system is the so-called deduction theorem. It enables us to move formulas from the context into the conclusion and shows a connection between context and implications.

Deduction theorem

Fix a context \(\Gamma\) and a formula \(\mathfrak{a}\). If \(\Gamma \cup\{\mathfrak{a}\} \vdash \mathfrak{b}\) then \(\Gamma \vdash \mathfrak{a} \to \mathfrak{b}\).

We fix \(\Gamma\) and \(\mathfrak{a}\), and use the induction on the set of consequences of \(\Gamma \cup\{\mathfrak{a}\}\). This means that if \(\mathfrak{b}\) is in the set of consequences, it has the form of one of the three cases mentioned earlier:
  1. \(\mathfrak{b} \in \Gamma \cup \{\mathfrak{a}\}\)
  2. \(\mathfrak{b}\) is an axiom schema
  3. There is proposition \(\mathfrak{h}\) such that \(\Gamma \cup\{\mathfrak{a}\} \vdash \mathfrak{h}\) and \(\Gamma \cup\{\mathfrak{a}\} \vdash \mathfrak{h} \to \mathfrak{b}\)

Case 1
If \(\mathfrak{b} \in \Gamma\), we know that \(\Gamma \vdash \mathfrak{b}\) from a definition of a proof. Using the first axiom schema we also know that \(\Gamma \vdash \mathfrak{b} \to (\mathfrak{a} \to \mathfrak{b})\). The conclusion \(\Gamma \vdash \mathfrak{a} \to \mathfrak{b}\) is then obtained by modus ponens. If \(\mathfrak{b}\) is equivalent to \(\mathfrak{a}\), we need to show \(\Gamma \vdash \mathfrak{b} \to \mathfrak{b}\). But we already proved this.

Case 2
If \(\mathfrak{b}\) is an axiom schema, we know that \(\Gamma \vdash \mathfrak{b}\) from definition of proof. We use the first axiom schema to conclude \(\Gamma \vdash \mathfrak{b} \to (\mathfrak{a} \to \mathfrak{b})\) and then use the modus ponens to obtain the conclusion \(\Gamma \vdash \mathfrak{a} \to \mathfrak{b}\).

Case 3
Using the inductive hypothesis we to obtain \(\Gamma \vdash \mathfrak{a} \to \mathfrak{h}\) and \(\Gamma \vdash \mathfrak{a} \to (\mathfrak{h} \to \mathfrak{b})\) from \(\Gamma \cup\{\mathfrak{a}\} \vdash \mathfrak{h}\) and \(\Gamma \cup\{\mathfrak{a}\} \vdash \mathfrak{h} \to \mathfrak{b}\). We use the second axiom schema to obtain \(\Gamma \vdash (\mathfrak{a} \to (\mathfrak{h} \to \mathfrak{b})) \to ((\mathfrak{a} \to \mathfrak{h}) \to (\mathfrak{a} \to \mathfrak{b}))\). Using the modus ponens twice, we conclude \(\Gamma \vdash \mathfrak{a} \to \mathfrak{b}\).

Natural deduction

The Hilbert style is not the only way to set up a logic system. The axiomatic systems have mainly axioms and only one rule. What if we had only one axiom and many rules? The idea is to shift the axiomatic system from formulas to logical symbols. Let's look at the conjunction. The Hilbert system had an axiom which introduces the proposition involving conjunction, namely the third axiom \(\mathfrak{a} \to (\mathfrak{b} \to (\mathfrak{a} \land \mathfrak{b}))\). If, for some context \(\Gamma\), we know that \(\Gamma \vdash \mathfrak{a}\) and \(\Gamma \vdash \mathfrak{b}\), we can use modus ponens two times to obtain \(\Gamma \vdash \mathfrak{a} \land \mathfrak{b}\). Instead of having this axiom in our system, we will work with a rule describing how to obtain a conjunction:

$$\dfrac{\Gamma \vdash \mathfrak{a} \quad \Gamma \vdash \mathfrak{b}}{\Gamma \vdash \mathfrak{a} \land \mathfrak{b}}$$

The part above the line shows what we assume and the lower part shows what we can build given the assumptions. This is in agreement with the axiom schema where the modus ponens is built into the introduction rule itself. We can also break apart the conjunction in two ways. This is what the fourth and fifth axiom schemas tell us. In order to completely describe the behaviour of conjunction, we end up with one introduction rule and two elimination rules.

$$\dfrac{\Gamma \vdash \mathfrak{a} \qquad \Gamma \vdash \mathfrak{b}}{\Gamma \vdash \mathfrak{a} \land \mathfrak{b}} \land_I \quad \dfrac{\Gamma \vdash \mathfrak{a} \land \mathfrak{b}}{\Gamma \vdash \mathfrak{a}} \land_E \quad \frac{\Gamma \vdash \mathfrak{a} \land \mathfrak{b}}{\Gamma \vdash \mathfrak{b}} \land_E$$

Implication has one introduction rule and one elimination rule. Remember that we use modus ponens with implication and the premise to obtain the conclusion. This means that the introduction rule for implication has to be the modus ponens. We have also seen how to create implication by reducing the context set. The elimination rule is thus expressed by the deduction theorem.

$$\dfrac{\Gamma \cup\{\mathfrak{a}\} \vdash \mathfrak{b}}{\Gamma \vdash \mathfrak{a} \to \mathfrak{b}} \to_I \qquad \frac{\Gamma \vdash \mathfrak{a} \to \mathfrak{b} \quad \Gamma \vdash \mathfrak{a}}{\Gamma \vdash \mathfrak{b}} \to_E$$

Next logical connective is the disjunction. The two introduction rules are modelled according to the sixth and seventh axiom. The elimination rule mirrors the eighth axiom.

$$\dfrac{\Gamma \vdash \mathfrak{a}}{\Gamma \vdash \mathfrak{a} \lor \mathfrak{b}} \lor_I \qquad \dfrac{\Gamma \vdash \mathfrak{b}}{\Gamma \vdash \mathfrak{a} \lor \mathfrak{b}} \lor_I \qquad \dfrac{\Gamma \vdash \mathfrak{a} \lor \mathfrak{b} \quad \Gamma \cup\{\mathfrak{a}\} \vdash \mathfrak{c} \quad \Gamma \cup\{\mathfrak{b}\} \vdash \mathfrak{c}}{\Gamma \vdash \mathfrak{c}} \lor_E$$

The next rule mirrors the ninth axiom schema and is called reductio ad absurdum. This is the rule we use when we write proofs by contradiction. There are logic systems which do not assume this rule like minimal or intuitionistic logic. By using this rule, we are working with the so-called classical propositional logic.

$$\frac{\Gamma \cup\{\lnot \mathfrak{a}\} \vdash \bot}{\Gamma \vdash \mathfrak{a}} \bot_E$$

The last rule is called the assumption rule. Remember that we proved \(\mathfrak{a} \to \mathfrak{a}\) from the first two axiom schemas. There is nothing above the line which means we do not have to assume anything to use this rule. This rule introduces assumptions.

$$\frac{} {\Gamma \cup \{\mathfrak{a}\} \vdash \mathfrak{a}}$$

It seems that we encoded all the axiom schemas into introduction or elimination rules but the first two. We will prove the first axiom schema by using natural deduction to show that we can recover this axiom schema. I won't state the definition of proof in natural deduction as it would be rather involved. It is essentially a rooted tree of introduction and elimination rules with root being the conclusion. So how does the proof of the first axiom schema look like? We use the elimination rule twice together with the rule for introducing assumptions.

$$\dfrac{\dfrac{\dfrac{} {\Gamma \cup\{\mathfrak{a},\mathfrak{b}\} \vdash \mathfrak{a}}} {\Gamma \cup\{\mathfrak{a}\} \vdash \mathfrak{b} \to \mathfrak{a}} \to_I} {\Gamma \vdash \mathfrak{a} \to (\mathfrak{b} \to \mathfrak{a})} \to_I$$

The second axiom schema can be proved too and you can try it yourself. The following definition speaks about the set of all possible sequents we can prove. It is an inductively defined set in the same way the set of formulas over some language. This means we can use induction to prove properties about this set.

The set of sequents provable in natural deduction is a set of sequents inductively genereated by the introduction and elimination rules of natural deduction.

One such use of induction over the set of sequents is the following lemma called weakening. It tells us that the natural deduction is a monotone system. This means that adding a hypothesis does not destroy propositions which were earlier provable.

If \(\Gamma \vdash \mathfrak{a}\) holds in natural deduction and \(\Gamma \subset \Delta\), then \(\Delta \vdash \mathfrak{a}\) holds in natural deduction.
We will use induction on the set of provable sequents.

Case of assumption
Let's say the formula \(\mathfrak{a}\) was proved by the assumption rule from the context \(\Gamma\). This means we have some set \(\Omega\) such that \(\Gamma = \Omega \cup \{\mathfrak{a}\}\). What this means is that \(\mathfrak{a}\) is in \(\Gamma\) and therefore in \(\Delta\). We can use the assumption rule to get \(\Delta \cup \{\mathfrak{a}\} \vdash \mathfrak{a}\) and because \(\mathfrak{a}\) is in \(\Delta\), we finally get \(\Delta \vdash \mathfrak{a}\).

Case of conjunction introduction
Suppose that \(\Gamma \vdash \mathfrak{a} \land \mathfrak{b}\). This means that \(\Gamma \vdash \mathfrak{a}\) and \(\Gamma \vdash \mathfrak{b}\) hold too. Using induction on the formulas \(\mathfrak{a}\) and \(\mathfrak{b}\), we get that \(\Delta \vdash \mathfrak{a}\) and \(\Delta \vdash \mathfrak{b}\). From the introduction rule for conjunction we conclude that \(\Delta \vdash \mathfrak{a} \land \mathfrak{b}\).

Other cases are done similarly. To write a full proof, you would need to write down the justification for all eleven rules. This could be done but it is not very interesting.

The weakening theorem is not something which is valid for all logical systems. For example the Believe revision is a process of changing beliefs to accomodate a new piece of information \(\mathfrak{m}\). This piece of information might be inconsistent with the old set of beliefs expressed by the context \(\Gamma\), and we might need to adjust the set of beliefs to maintain consistency. In other words, it is not true that \(\Gamma \vdash \mathfrak{a}\) automatically implies \(\Gamma \cup \{\mathfrak{m}\}\vdash \mathfrak{a}\)
