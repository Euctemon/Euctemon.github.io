Truth tables versus proofs

An achingly important questing is whether there is any connection between truth tables and things we can prove. To answer this question for the propositional logic, we need a definition of what it means to deduce a formula from a set of formulas and their truth tables. When we spoke about proof systems, we assumed that we can start with a set of formulas and called it a context. This context is also important when working with model theory and we often call such set of formulas as a theory.

Let \(\mathcal{L}\) be a language, and let \(\mathcal{T}\) be a theory and \(\mathcal{M}\) a model in \(\mathcal{L}\). We say that the model \(\mathcal{M}\) satisfies \(\mathcal{T}\), if for every formula \(\mathfrak{f} \in \mathcal{T}\), it follows that \(\mathcal{M} \vDash \mathfrak{f}\). We say that \(\mathfrak{a}\) is a semantic consequence of \(\mathcal{T}\), if every model of \(\mathcal{T}\) is also a model for \(\mathfrak{a}\). We write this as \(\mathcal{T} \vDash \mathfrak{a}\).

Note that \(\mathcal{M} \vDash \mathfrak{f}\) means that for every assignment function, the formula is satisfiable in the model. We called such formulas true. Also note that a theory in first order logic is a set of sentences, where a sentence is a formula without so called free variables. It means that all variables in a formula are quantified by the universal or existential quantifier.

The main goal of the rest of the chapter is to show the connection between \(\vdash\) and \(\vDash\) for propositional logic. We cannot prove it for any logical system as the interpretation of the truth symbol \(\vDash\) is tied to a given model and provability symbol \(\vdash\) to a given proof system.

Soundness

A logical system is sound if every theory \(\mathcal{T}\), a formula provable from \(\mathcal{T}\) is also a semantic consequence of \(\mathcal{T}\). In the case of propositional logic, this implication is quite simple to prove. Remember that for a propositional logic, we assume the one and only true model made of two elements representing truth and falsity. The proofs in this chapter will therefore not specify in which model we work.

For every theory \(\mathcal{T}\) and every propositional formula \(\mathfrak{a}\), \(\mathcal{T} \vdash \mathfrak{a}\) implies \(\mathcal{T} \vDash \mathfrak{a}\).
We use the induction on the set of sequents in natural deduction for the set \(\mathcal{T}\). We prove the theorem for three of the eleven rules for natural deduction as the proofs are similar.

Case of assumption
Suppose the proof of formula \(\mathfrak{a}\) was obtained by the assumption rule. This means that \(\mathfrak{a}\) is in \(\mathcal{T}\). From a definition of semantic consequence, we get \(\mathcal{T} \vDash \mathfrak{a}\).

Case of conjunction introduction
Suppose we derive \(\mathcal{T} \vdash \mathfrak{a} \land \mathfrak{b}\) from \(\mathcal{T} \vdash \mathfrak{a}\) and \(\mathcal{T} \vdash \mathfrak{b}\) using the introduction rule. Let \(v\) be any truth assignment which satisfies \(\mathcal{T}\). We use the inductive hypothesis to get \(\mathcal{T} \vDash \mathfrak{a}\) and \(\mathcal{T} \vDash \mathfrak{b}\). As \(v\) satisfies \(\mathcal{T}\), we know that \(v\) satisfies \(\mathfrak{a}\) and \(\mathfrak{b}\) from the definition of semantic consequence. We write this as \(\mathfrak{a}[v] = \top\) and \(\mathfrak{b}[v] = \top\). Using the definition of interpretation function, we find out that \((\mathfrak{a} \land \mathfrak{b}) [v] = \top\). In other words, we proved \(\mathcal{T} \vDash \mathfrak{a} \land \mathfrak{b}\).

Case of implication introduction
Suppose we derive \(\mathcal{T} \vdash \mathfrak{a} \to \mathfrak{b}\) from \(\mathcal{T} \cup \{\mathfrak{a}\} \vdash \mathfrak{b}\). Let \(v\) be any truth assignment satisfying \(\mathcal{T}\). Now it can happen that \(\mathfrak{a}[v] = \bot\) or \(\mathfrak{a}[v] = \top\). The first case implies \((\mathcal{a} \to \mathcal{b})[v] = \top\) from the definition of the interpretation function. If \(\mathcal{a}\) evaluates to true under the assignment \(v\), then the assignment function satisfies all formulas in \(\mathcal{T} \cup \{\mathfrak{a}\}\). From the induction hypothesis we know that \(\mathcal{T} \cup \{\mathfrak{a}\} \vDash \mathfrak{b}\). From the definition of semantic consequence we get \(\mathfrak{b}[v] = \top\). Because the assignment function satisfies both formulas, we get \((\mathfrak{a} \to \mathfrak{b})[v] = \top\) and thus \(\mathcal{T} \vDash \mathfrak{a} \to \mathfrak{b}\).

Completeness

A logical system is complete if, for every theory \(\mathcal{T}\), any semantic consequence \(\mathfrak{a}\) of \(\mathcal{T}\) is also provable from \(\mathcal{T}\). Proving soundness was relatively easy. We used introduction and elimination rules in natural deduction to show the provability relation preserves the truth assignment. Proving completeness means that we know that every truth assignment satisfying \(\mathcal{T}\) also satisfies \(\mathfrak{a}\), and we want to prove that there exists a proof for \(\mathfrak{a}\).

First thing we do is to introduce the notion of consistency. In the end, we will show that proving completeness is equivalent to showing that if the set \(\mathcal{T}\) is consistent, then is satisfiable. The first lemma shows what happens if the set \(\mathcal{T}\) is inconsistent.

Let \(\mathcal{T}\) be a theory. We say that \(\mathcal{T}\) is inconsistent if it proves falsity and consistent otherwise.
  1. The theory \(\mathcal{T}\) is inconsistent.
  2. For some formula \(\mathfrak{a}\), we have that \(\mathcal{T} \vdash \mathfrak{a}\) and \(\mathcal{T} \vdash \lnot \mathfrak{a}\) are provable.
  3. For every formula \(\mathfrak{a}\), we know that \(\mathcal{T} \vdash \mathfrak{a}\) is provable.

1 \(\to\) 3
Suppose that the set \(\mathcal{T}\) is inconsistent. This means that \(\mathcal{T} \vdash \bot\). Using weakening we can write \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \bot\) and using reductio ad absurdum we conclude \(\mathcal{T} \vdash \mathfrak{a}\).

3 \(\to\) 2
Suppose we can prove any propositional formula \(P\) from the set \(\mathcal{T}\). This means we can prove \(\mathcal{T} \vdash \mathfrak{a}\) and also \(\mathcal{T} \vdash \lnot \mathfrak{a}\) which we wanted.

2 \(\to\) 1
Suppose we can prove \(\mathcal{T} \vdash \mathfrak{a}\) and also \(\mathcal{T} \vdash \lnot \mathfrak{a}\). We know that \(\lnot \mathfrak{a}\) is an abbreviation for \(\mathfrak{a} \to \bot\) and using elimination rule for implication we conclude \(\mathcal{T} \vdash \bot\). This means \(\mathcal{T}\) is inconsistent.

We see that if \(\mathcal{T}\) is inconsistent, we can prove everything. This is called the principle of explosion. Being able to prove everything means that we lose the ability to speak about truth. If there are no false statements this means the proof of any statement carries no value at all.

Let \(\mathcal{T}\) be a theory. The theory \(\mathcal{T}\) is unsatisfiable if and only if \(\mathcal{T} \vDash \bot\).
If the falsity is a semantic consequence of a theory \(\mathcal{T}\), then this means that every model for \(\mathcal{T}\) is also a model for falsity. From the definition of satisfiability there is no model for falsity and therefore for the theory \(\mathcal{T}\).

The following lemma expresses that if we add a negation of a formula \(\mathfrak{a}\) to a theory \(\mathcal{T}\) which proves \(\mathfrak{a}\), we get inconsistency.

Let \(\mathcal{T}\) be any set of propositional formulas and let \(\mathfrak{a}\) be a propositional formula. Then \(\mathcal{T} \vdash \mathfrak{a}\) if and only if \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is inconsistent.
Suppose we can prove \(\mathcal{T} \vdash \mathfrak{a}\). By weakening we know \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \mathfrak{a}\) and by the assumption rule we also know \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \lnot \mathfrak{a}\). Using elimination rule for implication we get \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \bot\) and thus inconsistency. For the other implication assume that \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is inconsistent. This means we can prove \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \bot\). Using reductio ad absurdum we get \(\mathcal{T} \vdash \mathfrak{a}\).

We will now focus on the so-called maximally consistent sets (or theories). These are the set of propositions, which are in some sense saturated, meaning you cannot add another proposition without breaking consistency. Suprisingly, it is easier to prove that maximally consistent sets are satisfiable rather than proving satisfiability for mere consistent sets. First we establish that we can extend any consistent theory into a maximally consistent one.

A theory \(\mathcal{T}\) is maximally consistent if it is consistent, and for every formula \(\mathfrak{a}\) not in \(\mathcal{T}\), the set \(\mathcal{T} \cup \{\mathfrak{a}\}\) is inconsistent.
Let \(\mathcal{T}\) be any consistent theory. Then there exists a maximally consistent theory \(\Delta\) such that \(\mathcal{T} \subseteq \Delta\).
We assume that we have countably many propositional variables and therefore countably many formulas. This means we can enumerate these formulas as a sequence \(\mathfrak{a}_0,\mathfrak{a}_1,\ldots\) using natural numbers. We define a sequence of sets \(\Gamma_i\) inductively. We set \(\Gamma_0 = \mathcal{T}\) and then$$\Gamma_i = \begin{cases} \Gamma_i \cup \{\mathfrak{a}_i\} &\text{if such set is consistent} \\ \Gamma_i &\text{otherwise} \end{cases}$$We set \(\Delta = \bigcup \Gamma_i\) where the \(i\) ranges over natural numbers. By induction every \(\Gamma_i\) is consistent and thus \(\Delta\) is consistent too. We want to prove that \(\Delta\) is maximally consistent. Suppose that \(\mathfrak{c} \notin \Delta\). We know that \(\mathfrak{c} = \mathfrak{a}_i\) for some \(i\). We also know that \(\mathfrak{c}\) is not in \(\Gamma_{i+1}\) from the inductive construction. This means that \(\Gamma_{i+1} \cup \{\mathfrak{c}\}\) is inconsistent which implies that \(\Delta \cup \{\mathfrak{c}\}\) is inconsistent.

Before we prove that the maximally consistent theory is satisfiable, we need one more technical lemma. It tells us something about the structure of propositions one can find in a maximally consistent theory.

Let \(\mathcal{T}\) be a maximally consistent theory of propositional formulas. Then \(\mathcal{T}\) has the following properties:
  1. If \(\mathcal{T} \vdash \mathfrak{a}\), then \(\mathfrak{a} \in \mathcal{T}\).
  2. \(\bot \notin \mathcal{T}\)
  3. \(\mathfrak{a} \land \mathfrak{b} \in \mathcal{T}\) if and only if \(\mathfrak{a} \in \mathcal{T}\) and \(\mathfrak{b} \in \mathcal{T}\)
  4. \(\mathfrak{a} \to \mathfrak{b} \in \mathcal{T}\) if and only if \(\mathfrak{a} \notin \mathcal{T}\) or \(\mathfrak{b} \in \mathcal{T}\)
  5. \(\mathfrak{a} \lor \mathfrak{b} \in \mathcal{T}\) if and only if \(\mathfrak{a} \in \mathcal{T}\) or \(\mathfrak{b} \in \mathcal{T}\)

Case 1
Suppose that \(\mathcal{T} \vdash \mathfrak{a}\). Because \(\mathcal{T}\) is a consistent theory, we have that \(\mathcal{T} \cup \{\mathfrak{a}\}\) is consistent too. If you are not sure why, then think what would it mean if this set wasn't consistent. Inconsistency means we would be able to prove \(\mathcal{T} \cup \{\mathfrak{a}\} \vdash \bot\). Using introduction rule for implication we would then get \(\mathcal{T} \vdash \mathfrak{a} \to \bot\) which is the same as \(\mathcal{T} \vdash \lnot \mathfrak{a}\). From the first lemma in the previous section, this would mean that \(\mathcal{T}\) is inconsistent.

Case 2
The second claim follows from the fact that \(\mathcal{T}\) is consistent.

Case 3
Suppose that \(\mathfrak{a} \land \mathfrak{b} \in \mathcal{T}\). This means that \(\mathcal{T} \vdash \mathfrak{a}\) and \(\mathcal{T} \vdash \mathfrak{b}\) from the elimination rules of conjunction. Using the first case, this means that \(\mathfrak{a} \in \mathcal{T}\) and \(\mathfrak{b} \in \mathcal{T}\).

Case 4
Suppose that \(\mathfrak{a} \to \mathfrak{b} \in \mathcal{T}\). If \(\mathfrak{a} \in \mathcal{T}\), then \(\mathcal{T} \vdash \mathfrak{b}\) using the elimination rule for implication and thus \(\mathfrak{b} \in \mathcal{T}\) using the first case. Now suppose that \(\mathfrak{a}\) is not in \(\mathcal{T}\) or that \(\mathfrak{b}\) is. In the first case we get that \(\mathcal{T} \cup \{\mathfrak{a}\}\) is inconsistent and that implies \(\mathcal{T} \cup \{\mathfrak{a}\} \vdash \mathfrak{b}\). This then implies \(\mathcal{T} \vdash \mathfrak{a} \to \mathfrak{b}\) and \(\mathfrak{a} \to \mathfrak{b} \in \mathcal{T}\) by the first case. In the second case we have \(\mathcal{T} \vdash \mathfrak{b}\). By weakening we have \(\mathcal{T} \cup \{\mathfrak{a}\} \vdash \mathfrak{b}\) and thus the implication is in \(\mathcal{T}\) too.

Case 5
The proof is similar to the fourth case.

Equipped with the previous lemma, we define an unusual assignment function on formulas in the maximally consistent theory. This assignment function essentially tracks which formulas are in a given theory.

Suppose that \(\mathcal{T}\) is maximally consistent. Define a truth assignment \(v\) by setting \(v(\mathfrak{a}) = \top\) if \(\mathfrak{a} \in \mathcal{T}\). Then for every propositional formula we have \(\mathfrak{a}[v] = \top\) if and only if \(\mathfrak{a} \in \mathcal{T}\).
We use the induction on propositional formulas. If \(\mathfrak{a}\) is a variable, we know \(v(\mathfrak{a}) = \top\) because of how we defined the truth assignment. The cases for conjunction, disjunction and implication follow from the previous lemma.

We can finally prove that every consistent theory is satisfiable.

Every consistent theory \(\mathcal{T}\) is satisfiable.
Suppose \(\mathcal{T}\) is consistent. We can extend this set to a maximally consistent set \(\Delta\) using the inductive construction. The previous lemma tells us that we know how to define a truth assignment satisfying \(\Delta\). We know that \(\mathcal{T} \subset \Delta\), which implies that this assignment also satisfies \(\mathcal{T}\). Thus \(\mathcal{T}\) is satisfiable.

We have everything we need for proving that consistency is equivalent to completeness in classical propositional logic. By showing these two statements are equivalent, we prove the claim about completeness as we already proved consistency.

Every consistent theory \(\mathcal{T}\) of propositional formulas is satisfiable if and only if such theory is complete.

First assume that the consistency theorem holds and suppose that \(\mathcal{T} \nvdash \mathfrak{a}\). This implies that \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is consistent by a little lemma we proved in the beginning. By the consistency theorem we get that \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is satisfiable. Let \(v\) be a truth assignment which satisfies \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\). Such \(v\) must satisfy \(\lnot \mathfrak{a}\). Noting that negation is just disguised implication, this means that \(v\) satisfies \(\mathfrak{a} \to \bot\) and thus \(\mathfrak{a}[v] = \bot\). This means that \(\mathcal{T} \nvDash \mathfrak{a}\).

Now assume that the completeness theorem holds. Suppose that \(\mathcal{T}\) is consistent. From definition of consistency we have \(\mathcal{T} \nvdash \bot\). From the completeness theorem we have \(\mathcal{T} \nvDash \bot\). Using the second lemma in this section we conclude that \(\mathcal{T}\) is satifiable.

We proved that for classical propositional logic, proofs and truth tables go hand in hand. This has an enormous application in looking for proofs mechanically. Instead of trying to use natural deduction or other axiomatic system, we use powerful computers and truth tables to find whether a formula is a semantic consequence and therefore provable in a given theory. The programs used for that are called as satisfiability solvers, or commonly known as SAT solvers. These program employ clever tricks to avoid constructing the whole \(2^n\) rows of the truth table and nowadays are able to solve formulas with over thousands of variables.
