Truth tables versus proofs

An achingly important questing is whether there is any connection between truth tables and things we can prove. To answer this question for the propositional logic, we need a definition of what it means to deduce a formula from a set of formulas and their truth tables. When we spoke about proof systems, we assumed that we can start with a set of formulas and called it a context. This context is also important when working with model theory and we often call such set of formulas as a theory.

Let \(\mathcal{L}\) be a language, and let \(\mathcal{T}\) be a theory and \(\mathcal{M}\) a model in \(\mathcal{L}\). We say that the model \(\mathcal{M}\) satisfies \(\mathcal{T}\), if for every formula \(\mathfrak{f} \in \mathcal{T}\), it follows that \(\mathcal{M} \vDash \mathfrak{f}\). We say that \(\mathfrak{a}\) is a semantic consequence of \(\mathcal{T}\), if every model of \(\mathcal{T}\) is also a model for \(\mathfrak{a}\). We write this as \(\mathcal{T} \vDash \mathfrak{a}\).

Note that \(\mathcal{M} \vDash \mathfrak{f}\) means that for every assignment function, the formula is satisfiable in the model. We called such formulas true. Also note that a theory in first order logic is a set of sentences, where a sentence is a formula without so called free variables. It means that all variables in a formula are quantified by the universal or existential quantifier.

The main goal of the rest of the chapter is to show the connection between \(\vdash\) and \(\vDash\) for propositional logic. We cannot prove it for any logical system as the interpretation of the truth symbol \(\vDash\) is tied to a given model and provability symbol \(\vdash\) to a given proof system.

Soundness

A logical system is sound if every theory \(\mathcal{T}\), a formula provable from \(\mathcal{T}\) is also a semantic consequence of \(\mathcal{T}\). In the case of propositional logic, this implication is quite simple to prove.

For every theory \(\mathcal{T}\) and every propositional formula \(\mathfrak{a}\), \(\mathcal{T} \vdash \mathfrak{a}\) implies \(\mathcal{T} \vDash \mathfrak{a}\).

Before we get to the proof, remember that for a propositional logic, we assume the one and only true model made of two elements representing truth and falsity. The proofs in this chapter will therefore not specify in which model we work.

We use the induction on the set of sequents in natural deduction for the set \(\mathcal{T}\). We prove the theorem for three of the eleven rules for natural deduction as the proofs are similar.

Case of assumption
Suppose the proof of formula \(\mathfrak{a}\) was obtained by the assumption rule. This means that \(\mathfrak{a}\) is in \(\mathcal{T}\). From a definition of semantic consequence, we get \(\mathcal{T} \vDash \mathfrak{a}\).

Case of conjunction introduction
Suppose we derive \(\mathcal{T} \vdash \mathfrak{a} \land \mathfrak{b}\) from \(\mathcal{T} \vdash \mathfrak{a}\) and \(\mathcal{T} \vdash \mathfrak{b}\) using the introduction rule. Let \(v\) be any truth assignment which satisfies \(\mathcal{T}\). We use the inductive hypothesis to get \(\mathcal{T} \vDash \mathfrak{a}\) and \(\mathcal{T} \vDash \mathfrak{b}\). As \(v\) satisfies \(\mathcal{T}\), we know that \(v\) satisfies \(\mathfrak{a}\) and \(\mathfrak{b}\) from the definition of semantic consequence. We write this as \(\mathfrak{a}[v] = \top\) and \(\mathfrak{b}[v] = \top\). Using the definition of interpretation function, we find out that \((\mathfrak{a} \land \mathfrak{b}) [v] = \top\). In other words, we proved \(\mathcal{T} \vDash \mathfrak{a} \land \mathfrak{b}\).

Case of implication introduction
Suppose we derive \(\mathcal{T} \vdash \mathfrak{a} \to \mathfrak{b}\) from \(\mathcal{T} \cup \{\mathfrak{a}\} \vdash \mathfrak{b}\). Let \(v\) be any truth assignment satisfying \(\mathcal{T}\). Now it can happen that \(\mathfrak{a}[v] = \bot\) or \(\mathfrak{a}[v] = \top\). The first case implies \((\mathcal{a} \to \mathcal{b})[v] = \top\) from the definition of the interpretation function. If \(\mathcal{a}\) evaluates to true under the assignment \(v\), then the assignment function satisfies all formulas in \(\mathcal{T} \cup \{\mathfrak{a}\}\). From the induction hypothesis we know that \(\mathcal{T} \cup \{\mathfrak{a}\} \vDash \mathfrak{b}\). From the definition of semantic consequence we get \(\mathfrak{b}[v] = \top\). Because the assignment function satisfies both formulas, we get \((\mathfrak{a} \to \mathfrak{b})[v] = \top\) and thus \(\mathcal{T} \vDash \mathfrak{a} \to \mathfrak{b}\).

Completeness

Consistency

The completeness deals with the other way, that is, if we know \(\mathcal{T} \vDash P\), can we conclude that \(\mathcal{T} \vdash P\)? Proving soundness was relatively easy. We used introduction and elimination rules in natural deduction to show the provability relation preserves the truth assignment. Now we know \(\mathcal{T} \vDash \mathfrak{a}\) which means that every truth assignment satisfying \(\mathcal{T}\) also satisfies \(\mathfrak{a}\) and want to say that there exists a proof of \(\mathfrak{a}\) from \(\mathcal{T}\) without actually constructing the proof. That is not trivial at all! We start with the statement of the theorem.

For every set \(\mathcal{T}\) of propositional formulas and every propositional formula \(\mathfrak{a}\), we have that \(\mathcal{T} \vDash \mathfrak{a}\) implies \(\mathcal{T} \vdash \mathfrak{a}\).

First thing we do is to introduce the notion of consistency. With few simple lemmas, we will show that proving completeness is equivalent to showing that if the set \(\mathcal{T}\) is consistent, then is satisfiable. The first lemma shows what happens if the set \(\mathcal{T}\) is inconsistent.

Let \(\mathcal{T}\) be any set of propositional formulas. We say that \(\mathcal{T}\) is inconsistent if it proves \(\bot\) and consistent otherwise.

  1. The set \(\mathcal{T}\) is inconsistent.
  2. For some formula \(\mathfrak{a}\), we have that \(\mathcal{T} \vdash \mathfrak{a}\) and \(\mathcal{T} \vdash \lnot \mathfrak{a}\) are provable.
  3. For every formula \(\mathfrak{a}\), we know that \(\mathcal{T} \vdash \mathfrak{a}\) is provable.
  • \(1 \to 3 \: \) Suppose that the set \(\mathcal{T}\) is inconsistent. This means \(\mathcal{T} \vdash \bot\). Using weakening we can write \(\mathcal{T} \cup \{\lnot P\} \vdash \bot\) and using reductio ad absurdum we conclude \(\mathcal{T} \vdash P\).
  • \(3 \to 2 \: \) Suppose we can prove any propositional formula \(P\) from the set \(\mathcal{T}\). This means we can prove \(\mathcal{T} \vdash \mathfrak{a}\) and also \(\mathcal{T} \vdash \lnot \mathfrak{a}\) which we wanted.
  • \(2 \to 1 \:\) suppose we can prove \(\mathcal{T} \vdash \mathfrak{a}\) and also \(\mathcal{T} \vdash \lnot \mathfrak{a}\). We know that \(\lnot \mathfrak{a}\) is an abbreviation for \(\mathfrak{a} \to \bot\) and using elimination rule for implication we conclude \(\mathcal{T} \vdash \bot\). This means \(\mathcal{T}\) is inconsistent.

\(\blacksquare\)

We see that if \(\mathcal{T}\) is inconsistent, then we can prove everything. This is called the principle of explosion. Being able to prove everything means that we lose the ability to speak about truth. If there are no false statements this means the proof of any statement carries no value at all. The next lemma is a simple observation that we cannot find a truth assignment which satisfies \(\bot\). The proof is trivial as we only have to carefully write down the definition of satisfiability and remember that \(I(\bot\) = \bot\). We used this line of reasoning for the case of reductio ad absurdum in the proof of soundness.

Let \(\mathcal{T}\) be any set of propositional formulas. The set \(\mathcal{T}\) is unsatisfiable if and only if \(\mathcal{T} \vDash \bot\).

What happens when we add a negation of a formula \(\mathfrak{a}\) to the set \(\mathcal{T}\) from which we could prove \(\mathfrak{a}\)? We get inconsistency. This innocently looking lemma will be of much importance later on for using SAT solvers for proving. It is also important for the completeness theorem.

Let \(\mathcal{T}\) be any set of propositional formulas and let \(\mathfrak{a}\) be a propositional formula. Then \(\mathcal{T} \vdash \mathfrak{a}\) if and only if \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is inconsistent.

Suppose we can prove \(\mathcal{T} \vdash \mathfrak{a}\). By weakening we know \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \mathfrak{a}\) and by the assumption rule we also know \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \lnot \mathfrak{a}\). Using elimination rule for implication we get \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \bot\). This means that \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is inconsistent. For the other implication assume that \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is inconsistent. This means we can prove \(\mathcal{T} \cup \{\lnot \mathfrak{a}\} \vdash \bot\). Using reductio ad absurdum we immidately get \(\mathcal{T} \vdash \mathfrak{a}\).

\(\blacksquare\)

We are getting closer to the proof of completeness. The next theorem does not have a name, but we will call it Consistency theorem. We prove that Consistency theorem is in classical propositional logic equivalent to the Completeness theorem. This means we can prove Completeness theorem using Consistency theorem and vice versa.

Every consistent set \(\mathcal{T}\) of propositional formulas is satisfiable.

Of equivalence

First assume that Consistency theorem holds. Suppose that \(\mathcal{T} \n+ nvdash \mathfrak{a}\). From the previous lemma we know \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is consistent. By the Consistency theorem we know that \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\) is satisfiable. Let \(v\) be a truth assignment which satisfies \(\mathcal{T} \cup \{\lnot \mathfrak{a}\}\). Such \(v\) must satisfy \(\lnot \mathfrak{a}\), in other words \(\llbracket \mathfrak{a} \to \bot\rrbracket^v = \top\). This is possible only if \(\llbracket \mathfrak{a} \rrbracket^v = \bot\) from the definition of evaluating implication. Thus \(\mathcal{T} \nvDash \mathfrak{a}\).

Now assume that the Completeness theorem holds. Suppose that \(\mathcal{T}\) is consistent. From definition of consistency we have \(\mathcal{T} \n+ nvdash \bot\). From the Completeness theorem we have \(\mathcal{T} \nvDash \bot\). Using the first lemma in this section we conclude that \(\mathcal{T}\) is satifiable.

Maximally consistent sets

We are left with proving the Consistency theorem. In order to prove this statement, we will need to introduce maximally consistent sets. These are the set of propositions, which are in some sense saturated, meaning you cannot add another proposition without breaking consistency. Suprisingly, it will be easier to prove that maximally consistent sets are satisfiable and from that prove the Consistency theorem.

\mathfrak{a} set \(\mathcal{T}\) of propositional formulas is maximally consistent if

  • it is consistent, and
  • for every propositional formula \(\mathfrak{b} \notin \mathcal{T}\), the set \(\mathcal{T}, \mathfrak{b}\) is inconsistent.

First we need to establish that we can extend any consistent set into a maximally consistent one.

Let \(\mathcal{T}\) be any consistent set of propositional formulas. Then there exists a maximally consistent set \(\Delta\) such that \(\mathcal{T} \subset \Delta\).

We assume that we have countably many propositional variables and therefore countably many propositional formulas. This means we can enumerate these propositions as a sequence \(A_0,A_1,\ldots\) using natural numbers. We define a sequence of sets \(\Gamma_i\) inductively. We set \(\Gamma_0 = \mathcal{T}\) and

$$\Gamma_i = \begin{cases}\n \Gamma_i \cup \{A_i\} &\text{if such set is consistent} \\n \Gamma_i &\text{otherwise }\n \end{cases}$$

We set \(\Delta = \bigcup \Gamma_i\) where the \(i\) ranges over natural numbers. By induction every \(\Gamma_i\) is consistent and thus \(\Delta\) is consistent too. We want to prove that \(\Delta\) is maximally consistent. Suppose that \(\mathfrak{a} \notin \Delta\). We know that \(\mathfrak{a} = B_i\) for some \(i\). We also know that \(\mathfrak{a} \notin \Gamma_{i+1}\) from the inductive construction. This means that \(\Gamma_{i+1} \cup \{\mathfrak{a}\}\) is inconsistent which implies that \(\Delta \cup \{\mathfrak{a}\}\) is inconsistent.

\(\blacksquare\)

Before we prove that the maximally consistent set is satisfiable, we need one more technical lemma. It tells us something about the structure of propositions one can find in a maximally consistent set.

Let \(\mathcal{T}\) be a maximally consistent set of propositional formulas. Then \(\mathcal{T}\) has the following properties

  1. \(\mathcal{T}\) is closed under deduction, which means if \(\mathcal{T} \vdash \mathfrak{a}\), then \(\mathfrak{a} \in \mathcal{T}\).
  2. \(\bot \notin \mathcal{T}\)
  3. \(\mathfrak{a} \land \mathfrak{b} \in \mathcal{T}\) if and only if \(\mathfrak{a} \in \mathcal{T}\) and \(\mathfrak{b} \in \mathcal{T}\)
  4. \(\mathfrak{a} \to \mathfrak{b} \in \mathcal{T}\) if and only if \(\mathfrak{a} \notin \mathcal{T}\) or \(\mathfrak{b} \in \mathcal{T}\)
  5. \(\mathfrak{a} \lor \mathfrak{b} \in \mathcal{T}\) if and only if \(\mathfrak{a} \in \mathcal{T}\) or \(\mathfrak{b} \in \mathcal{T}\)

Suppose that \(\mathcal{T} \vdash \mathfrak{a}\). We assume that \(\mathcal{T}\) is consistent, which implies that \(\mathcal{T} \cup \{\mathfrak{a}\}\) is consistent too. If you are not sure why, then think what would it mean if this set wasn't consistent. Inconsistency means we are able to prove \(\mathcal{T},\mathfrak{a} \vdash \bot\). Using introduction rule for implication we get \(\mathcal{T} \vdash \mathfrak{a} \to \bot\) which is the same as \(\mathcal{T} \vdash \lnot \mathfrak{a}\). From the first lemma in the previous section, this would mean that \(\mathcal{T}\) is inconsistent! The second claim follow from the first one, again using the first lemma in the previous section.

Suppose that \(\mathcal{T}\) is maximally consistent. Define a truth assignment \(v\) by setting \(v(P) = \top\) if \(P \in \mathcal{T}\). Then for every propositional formula we have \(\llbracket \mathfrak{a} \rrbracket^v = \top\) if and only if \(\mathfrak{a} \in \mathcal{T}\).

We use the induction on propositional formulas \(\mathfrak{a}\). if \(\mathfrak{a}\) is a propositional variable then we know \(v(\mathfrak{a}) = \top\) because that is how we defined the truth assignment for variables. The cases for conjunction, disjunction and implication follow from the previous lemma.

\(\blacksquare\)

We can finally prove the consistency theorem. We repeat its statement and then prove it.

Every consistent set \(\mathcal{T}\) of propositional formulas is satisfiable.

Suppose \(\mathcal{T}\) is consistent. We can extend this set to a maximally consistent set \(\Delta\) using the inductive construction. The previous lemma tells us that we know how to define a truth assignment satisfying \(\Delta\). We know that \(\mathcal{T} \subset \Delta\), which implies that this assignment also satisfies \(\mathcal{T}\). Thus \(\mathcal{T}\) is satisfiable.

\(\blacksquare\)
