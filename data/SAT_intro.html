Introduction

These few upcoming pages are my attempt to formalize few definitions and theorem from basic abstract algebra. The goal is to start with monoids and work the way up to fields. The catch is that I will be using a theorem prover called LEAN a formalize all the mathematics in this software. You can think about a theorem prover as a software which enables you to write mathematics almost as a computer program. This text will be havily centered around the LEAN theorem prover. This means that if you are here just for the mathematics you might have troubles reading these pages as they will be filled with many code snippets. It takes a considerable amout of time to learn a software for theorem proving. I will try to explain the basic syntax as we go through the definitions and theorem but it is not in my power to teach you how to use LEAN on the spot.If however you find out that you would like to try it personally then the official site mentioned earlier contains the install and setup instructions. After that you might be interested in Mathematics in LEAN which is a website specially dedicated to teach you how to do mathematics in LEAN. 


                inductive Sum (α : Type u) (β : Type u) : Type u where
                  | inl : α → Sum α β
                  | inr : β → Sum α β
