Introduction

The main goal of this topic is to develop a working SAT solver. It is a program which aims to find a solution to a given boolean satisfiability problem. But what is a boolean satisfiability problem? Let's say we are given a formula $$(P \lor Q) \land R$$ and our task is to assign true of false value to all the variables in order to make this formula true. In order for the formula to be true, both parts of the and clause needs to be true. This specially means \(R\) needs to be true. For the or clause we can choose for example \(P\) to be true and \(Q\) to be false. This results in the or clause being true and therefore the whole formula. The goal of a SAT solver is do this process computationally. Before we start to talk about implementing a solver, we need to talk about the language of propositional logic. Next two chapters are devoted to soundness and completeness of classical propositional logic and its implications for using SAT solvers to find out whether a formula is provable from a set of hypotheses. After that we start implementing a naive solver and talk about if we can do better.
