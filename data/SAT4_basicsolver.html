Conjunctive normal form

When we defined the alphabet of propositional logic, we used the set \(\bot, \land, \lor, \to\). But modern solvers use different set as the alphabet, namely the set \(\bot, \top, \land, \lor\). You can see the implication was replaced by negation. One might wonder if we can do this. Do these two sets generate the same set of formulas? We need to be sure that we can generate formulas of type \(P \to Q\).
