Models

In this chapter we look at semantics of propositional logic. This has a clse connection to the truth tables you surely know. But in order to better understand why are truth tables a valuable tool, we need to talk about these things in more general terms. Semantics deals with models and their interpretations. You might remember that defined a model in the previous chapter. A model is a tuple containing a set called universe and a function \(I\) which assigns a function to a function symbol and relation to a relation symbol. For propositional logic we therefore need to specify the universe and function for every symbol in the alphabet as the alphabet is only made of functional symbols. We will also need a function \(v\) which assigns elements of the universe to propositional variables. This is commonly called truth assignment. For the universe we set \(\mathbb{B} = \{\bot,\top\}\). This has nothing to do with our alphabet. We could have chosen a set \(\{0,1\}\) or any other set of two distict symbols! We are ready to define the function \(I\)

Let \(v\) be a truth assignment. We interpret the function symbols as follows
  • \(I(P) = v(P)\) if \(P\) is propositional variable
  • \(I(\bot) = \bot\)
  • If \(I(P) = \top\) and \(I(Q) = \top\), then \(I(P \land Q) = \top\), otherwise \(I(P \land Q) = \bot\)
  • If \(I(P) = \top\) or \(I(Q) = \top\), then \(I(P \lor Q) = \top\), otherwise \(I(P \lor Q) = \bot\)
  • If \(I(P) = \bot\) or \(I(Q) = \top\), then \(I(P \to Q) = \top\), otherwise \(I(P \to Q) = \bot\)

The assigning function here is written little differently than the one used in the definition of natural number model. Nevertheless it is correct. For example the interpretation of conjunction, the function \(I(P \land Q)\) has the same arity as the conjunction symbol. Why? We need to supply two values, \(I(P)\) and \(I(Q)\) in order to evaluate this function and we know conjunction has the arity of two.

Let's take a look at some example. We fix three variables \(P,Q\) and \(R\) and write a truth assignment \(\langle P = \bot, Q = \top, R = \bot \rangle\). We can take a propositional formula with these variables and evaluate it using the function \(I\). Take for example the proposition \(P \to (Q \lor A)\). We then have \(I(Q \lor R) = \top\) and therefore \(I(P \to (Q \lor R)) = \top\). It is common to write the evaluation of proposition \(A\) using \(I\) given a truth assignment \(v\) as \(\llbracket A\rrbracket^v\). Now comes a crucial definition.

Let \(A\) be a propositional formula and let \(v\) be a truth assignment.
  • We say that \(v\) satisfies \(A\), written as \(v \models A\), if \(\llbracket A\rrbracket^v = \top\). We say that \(A\) is true under \(v\).
  • If \(\Gamma\) is a set of propositional formulas, we say \(v \models \Gamma\) if \(v \models A\) for every propositional formula \(A \in \Gamma\).
  • A propositional formula \(A\) is satisfiable if there exists truth assignment \(v\) such that \(v \models A\).
  • A propositional formula \(A\) is valid if it is true under any truth assignment.
  • A propositional formula \(A\) is a logical consequence of \(\Gamma\), written as \(\Gamma \models A\), if every truth assignment \(v\) which satisfies \(\Gamma\) also satisfies \(A\).
